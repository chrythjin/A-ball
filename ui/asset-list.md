# UI/아이콘 에셋 리스트

이 문서는 "더 볼 (The Ball)" 게임의 UI 및 아이콘 에셋을 정리한 목록입니다.
각 화면별로 필요한 에셋, 용도, 권장 사양을 포함합니다.

---

## 로비 화면 (scene-lobby)

| # | 에셋명 | 용도 | 권장 형식 | 권장 크기 | 비고 |
|---|--------|------|-----------|-----------|------|
| 1 | 게임 로고/타이틀 | "더 볼" 메인 타이틀 이미지 | PNG (투명배경) | 320x80 | 중앙 배치, 어두운 배경에서 잘 보이는 밝은 톤 |
| 2 | 시작 버튼 (기본) | 게임 시작 버튼 기본 상태 | PNG or SVG | 200x60 | 라운드 코너, 밝은 배경 |
| 3 | 시작 버튼 (호버) | 게임 시작 버튼 호버 상태 | PNG or SVG | 200x60 | 기본보다 밝거나 글로우 효과 |
| 4 | 로비 배경 | 로비 분위기 연출용 배경 | PNG or JPG | 480x640 | 어두운 톤, 게임 테마에 맞는 추상적 패턴 |

## 컬러 팔레트 참고 (현재 코드 기반)

| 용도 | 현재 색상 | Hex |
|------|-----------|-----|
| 배경 (body) | 매우 어두운 네이비 | `#0f0f1a` |
| 게임 컨테이너 | 어두운 네이비 | `#1a1a2e` |
| 버튼 기본 | 회보라 | `#4a4e69` |
| 버튼 호버 | 밝은 회보라 | `#9a8c98` |
| 공 색상 | 밝은 베이지 | `#f2e9e4` |
| 데드라인 | 빨강 | `#ff4d4d` |
| 적 HP 높음 | 초록 | `rgb(0,255,0)` |
| 적 HP 낮음 | 빨강 | `rgb(255,0,0)` |
| HP 텍스트 | 흰색 | `#ffffff` |

에셋 제작 시 이 팔레트와 조화를 이루도록 디자인하는 것을 권장합니다.

## 대안: 이미지 없이 꾸미는 방법

이미지를 직접 제작하기 어려운 경우 다음으로 대체 가능합니다:

- **CSS 그라디언트/박스 섀도우** - 버튼, 배경, 팝업 프레임
- **이모지/유니코드** - 스킬 아이콘 (⚡ 💥 🎱 🔥)
- **Canvas API** - 글로우(shadowBlur), 그라디언트 원, 파티클 시스템
- **무료 에셋 사이트** - kenney.nl, opengameart.org, itch.io/game-assets
